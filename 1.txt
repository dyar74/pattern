- синглтон - DB Connector, Logger
- фабричный метод -  для реализации например абстрактной фабрики
- абстрактная фабрика - построитель форм, для разных систем, в играх для создания рас
- фасад - для упрощения работы со сложными системами в клиентском коде
- адаптер - для реализации доступа к старомому ПО
- компоновщик -работа с объектами в виде древовидной структуры, пример SimpleXML и jQuery
- стратегия - изменение поведения во время выполнения, например в играх
